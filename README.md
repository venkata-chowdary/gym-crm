# GymDesk CRM Application

## Overview
GymDesk is a premium mobile CRM for gym owners, enabling member management, workout tracking, and admin tasks via a modern UI with authentication, profile settings, and a responsive dashboard.

## Features
- **User Authentication**: Secure login and signup flows with OTP verification.
- **Member Management**: Add, edit, and view gym members.
- **Profile & Settings**: Personalize appearance, notification preferences, and account details.
- **Modern UI**: Premium design system with glassmorphism, dynamic animations, and dark mode.
- **Cross‑Platform**: Built with React Native (or appropriate framework) for iOS and Android.

## Getting Started
### Prerequisites
- Node.js (v18 or later)
- npm or yarn
- Android Studio / Xcode for mobile emulators

### Installation
```bash
# Clone the repository
git clone <repository-url>
cd "Gym Desk v2"

# Install dependencies
npm install   # or yarn install
```

### Running the App
```bash
# Start the development server
npm run dev   # or yarn dev
```

The app will launch on the default simulator/emulator. Use a physical device if preferred.
